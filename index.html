PERSONAL PROJECTS

I have worked on a large variety of self-started projects ranging from web-scraping to predictive modeling. For one, I've created a variety of Twitter bots using the Twitter REST API, many of which utilize powerful machine learning techniques such as sentiment analysis (NLPK), markov chains, personality insights (IBM Bluemix), and more. For some of these Twitter bots, I've implemented MySQL databases from which my scripts retrieve information vital to the bots' functions. These scripts also automatically update their databases on their own. In addition to these projects utilizing the Twitter API, I've done some projects in web-scraping. Using Python's urllib and beautiful soup libraries, I've built a web-crawler that hops from site to site gathering and storing email addresses. This web-crawler also utilizes a local MySQL database. Of course, in addition to all of these projects, I also wrote from scratch (HTML & CSS) the very webpage you are currently viewing!
